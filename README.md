# EasyCommands
Library to ease the process of making commands

Maven import:
```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>


	<dependency>
	    <groupId>com.github.TrollsterCooleg</groupId>
	    <artifactId>EasyCommands</artifactId>
	    <version>83208819c8</version>
	</dependency>
```
